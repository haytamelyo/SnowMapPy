    User Guide¶

    Welcome to the SnowMapPy User Guide. This section provides in-depth explanations of all features and concepts.

    What You'll Learn¶

    This guide covers:

    Topic Description
    Algorithm Overview The science behind the gap-filling process
    Interpolation Methods Choosing the right gap-filling method
    Spatial Correction DEM-based snow adjustments
    Output Formats Understanding and using output data
    Memory Optimization Processing large datasets efficiently

    Processing Pipeline¶

    SnowMapPy implements a multi-stage processing pipeline:

    graph TB
    subgraph Input
        A[MODIS Terra MOD10A1]
        B[MODIS Aqua MYD10A1]
        C[SRTM DEM]
        D[Study Area Shapefile]
    end

    subgraph "Quality Control"
        E[Remove invalid pixels]
        F[Apply QA flags]
    end

    subgraph "Sensor Fusion"
        G[Combine Terra + Aqua]
        H[Priority: Terra > Aqua]
    end

    subgraph "Gap Filling"
        I[6-Day Moving Window]
        J[Spatial Correction]
        K[Temporal Interpolation]
    end

    subgraph Output
        L[Gap-Free Zarr Dataset]
    end

    A --> E
    B --> E
    E --> F
    F --> G
    G --> H
    H --> I
    C --> J
    D --> I
    I --> J
    J --> K
    K --> L
